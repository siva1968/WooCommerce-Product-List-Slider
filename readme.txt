== Description ==
Display WooCommerce products in a responsive slider on shop and category pages. Highly customizable and mobile-friendly.

== Installation ==
1. Upload the plugin files to the `/wp-content/plugins/woocommerce-product-list-slider/` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Configure settings under 'Settings > Product List Slider'.

== Frequently Asked Questions ==
= How do I change the slider speed? =
Go to 'Settings > Product List Slider' and adjust the speed in milliseconds.
